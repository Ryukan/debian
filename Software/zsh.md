# Zsh
Doc: https://scrivocodice.it/installare-zsh-e-ohmyzsh-su-linux/

## Installazione
```bash
sudo apt update
sudo apt install zsh
```

## Zsh ad default shell
```bash
chsh -s $(which zsh)
```

## Install Oh My ZSH
```bash
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
